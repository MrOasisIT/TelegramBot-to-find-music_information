# Music Info Telegram Bot:

Telegram-бот для поиска информации о музыкальных треках с использованием Last.fm API.

Бот позволяет получить основную информацию о треке: исполнителя, дату выхода, длительность, жанры (теги), а также обложку релиза.

Проект создан в учебных и портфолио-целях для практики работы с API, асинхронным Python и Telegram-ботами.

# Возможности:

-  Поиск информации о треке
-  Исполнитель
-  Дата выхода трека
-  Длительность
-  Жанры (теги Last.fm)
-  Обложка альбома / релиза
-  Безопасная обработка отсутствующих данных (если информации нет — бот не падает)

# Используемые технологии:

- Python 3
- aiogram
- Last.fm API
- Async / await
- REST API
